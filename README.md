# Sistema de Recomendação de Músicas com Grafos

Este projeto utiliza Neo4j para recomendar músicas com base no comportamento do usuário.

## Como funciona
- Usuários escutam e curtem músicas
- O sistema analisa conexões no grafo
- Sugere novas músicas automaticamente

## Arquivos
- dados.cypher → cria o banco
- consultas.cypher → faz recomendações
